# 百度莱茨狗买卖助手
- 该辅助工具可用于最新百度莱茨狗，提供不同稀有度自动买卖功能。能自动识别百度验证码，自动交易。
- 百度验证码接口为自己制作并搭建，准确率在90%以上，因负载较高，为了能长期提供服务，收取一点费用，感谢理解！！！
- key请通过微信获取，获取方式请看界面说明。
- 设置好价格与其他配置后，点启动即可自动买卖。
